# Data Warehouse and Analytics Project

Welcome to the ** Data Warehouse and Analytics Project** repository.
This project uses industry best pratices in data engineering and analytics in building a modern data warehouse with SQL Server, including ETL processes, data modelling and analytics. It demonstrates a comprehensive data warehousing and analytics solution, from building a data warehouse to generating actionable insights.
